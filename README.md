# Express Gold Investments

A backend API for the Express Gold Investments crypto investment platform, built with **Express.js + TypeScript**.

## About the Project
This is the server-side application for managing crypto investment features (users, transactions, admin controls, etc.). It was originally developed on Replit and is now deployed on Vercel.

## Tech Stack
- Node.js + Express
- TypeScript
- pnpm (package manager)

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- pnpm

### Installation
```bash
git clone https://github.com/Victexchairman/Express-Gold-investments1.git
cd Express-Gold-investments1
pnpm install
